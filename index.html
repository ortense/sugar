Elementos customizados com Polymer

"Uma tendência para o futuro" é a melhor definição que consigo pensar para custom elements.

Todo código escrito aqui deve usar o highlight-code-element, um componente reaproveitavel escrito em html com a ajuda do Polymer.

O componete é muito bom e eu realmente recomendo a utilização (:

Nesse caso tudo que eu fiz foi:

<!doctype html> <html lang="pt-br"> <head> <!-- css que estiliza o código --> <link rel="stylesheet" href="highlight.css"> <!-- nesse caso especificamente jQuery é uma dependencia, mas não é obrigatorio --> <script src="jquery.js"></script> <!-- biblioteca do polymer --> <script src="polymer.js"></script> <link rel="import" href="highlight.html"> <title>Document</title> </head> <body> <!-- chamar meu componente --> <highlight-code width="100%" height="auto"> <!-- todo o codigo fica aqui --> </highlight-code> </body> </html>

O unico cuidado para html é o escape do "<" e reiniciar a identação que pode ser visto no código fonte

Data Binding com angular JS

Contador em {{count}}

Isso é extremamente poderoso, a criação de uma controller por meio de uma função global e a total abstração da manipulação do DOM, são caracteristicas que me fazem pensar na utilização do Angular.js

Outro contador {{count}}

Outro ponto que gostei muito é a praticidade de reaproveitamento, posso criar varios "componentes" com o mesmo controller sem que um interfira no outro, talvez seja uma combinação poderosa pra o Polymer.

Minha Controller para esse exemplo:

function SugarCountCtrl($scope) { //o contador é uma propriedade do escopo dessa controller $scope.count = 0; //funções chamadas pelos botões $scope.plusOne = function(){ $scope.count++ }; $scope.lessOne = function(){ $scope.count-- }; return $scope; }

Feito isso foi só escrever minha "view", mais ou menos assim:

<!doctype html> <html lang="pt-br" ng-app> <head> <!-- biblioteca do Angular --> <script src="angular.js"></script> <title>Document</title> </head> <body> <div ng-controller="SugarCountCtrl"> <p><strong>Outro contador {{count}}</strong></p> <p> <button type="button" class="btn btn btn-success" ng-click="plusOne()">mais um</button> <button type="button" class="btn btn btn-warning" ng-click="lessOne()">menos um</button> </p> </div> </body> </html>
